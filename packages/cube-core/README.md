# cube-core

A symbolic dynamical system operating on the 3-dimensional hypercube graph.

This package contains:
- cube topology
- symbolic traversal
- scoring
- concept seeding
- attractor scans
- Hamiltonian path generation
- 3D visualization

## Run

PYTHONPATH=src python examples/run.py

## Atlas

PYTHONPATH=src python examples/concept_atlas.py

## Attractors
