# The Cube

The Cube is the root system.

It contains:
- cube-core: symbolic hypercube traversal engine
- field-core: adaptive multi-agent field runtime
- arctopia: governance application layer
- ecoarq: environmental sensing / verification layer
- realert: live signal / monitoring layer

## Structure

- `packages/cube-core` → symbolic dynamical engine
- `packages/field-core` → realtime adaptive field system
- `apps/arctopia` → governance application
- `apps/ecoarq` → ecological / verification application
- `apps/realert` → live alerting / runtime monitoring

## Philosophy

Cube is the physics.
ArcTopia is one major civilization-scale application built on top of it.
